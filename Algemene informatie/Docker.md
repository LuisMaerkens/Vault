# Docker stappen

- Clone het GitHub-project
    `git clone <link>`
    
- Ga naar de projectmap
    `cd <mapnaam>`
    
- Bouw de Docker image
    `docker build -t <naam> .`
    
- Start de container met poortkoppeling en eventuele extra opties
    `docker run -p <host-poort>:<container-poort> <naam>`
